# 开机尾灯双闪功能需求文档

## 1. 功能概述

在设备开机时，尾灯（LED4）执行红色双闪动画，营造赛车启动的仪式感。双闪完成后，尾灯恢复到用户上次设置的颜色状态。

## 2. 用户故事

### 2.1 作为用户，我希望开机时看到炫酷的尾灯双闪效果
**验收标准：**
- 开机时尾灯自动执行红色双闪动画
- 双闪效果清晰可见，有赛车启动的仪式感
- 双闪时间适中，不会让用户等待太久（约1秒）
- 双闪完成后自动恢复到用户设置的颜色

### 2.2 作为用户，我希望双闪效果配合开机流程自然流畅
**验收标准：**
- 双闪在LCD显示Logo之后执行
- 双闪不阻塞系统初始化
- 双闪完成后立即进入主界面
- 整个开机流程流畅自然

### 2.3 作为用户，我希望双闪不影响其他LED灯带的状态
**验收标准：**
- 双闪只影响尾灯（LED4）
- 其他LED灯带（LED1/2/3）保持正常状态
- 双闪完成后，所有LED恢复到用户设置

### 2.4 作为用户，我希望双闪效果可以配合音效（可选）
**验收标准：**
- 双闪时可以播放短促的引擎音效
- 音效与双闪节奏同步
- 音效不影响双闪的视觉效果

## 3. 功能需求

### 3.1 双闪效果参数

#### 3.1.1 颜色
- **主色**: 纯红色 RGB(255, 0, 0)
- **理由**: 红色是最经典的赛车警示色，视觉冲击力强

#### 3.1.2 闪烁模式
- **模式**: 快速闪烁（亮-灭-亮-灭-亮-灭）
- **次数**: 3次
- **频率**: 150ms亮 + 150ms灭 = 300ms/次
- **总时长**: 约900ms（不到1秒）

#### 3.1.3 亮度
- **亮度**: 全亮（255）
- **理由**: 开机动画需要足够的视觉冲击力

### 3.2 触发时机

#### 3.2.1 触发位置
- **文件**: `f4_26_1.1/f4_26_1.1/f4_26_1.1/Core/Src/main.c`
- **位置**: 在 `LCD_ui0()` 显示Logo之后，进入主循环之前

#### 3.2.2 执行顺序
```
1. 系统初始化
2. LCD显示Logo (LCD_ui0)
3. 【新增】尾灯双闪动画
4. 【可选】播放引擎音效
5. 恢复LED状态
6. 进入主循环
```

### 3.3 状态恢复

#### 3.3.1 LED状态保存
- 双闪开始前，保存当前LED4的颜色设置
- 从Flash读取用户上次设置的颜色（red4, green4, blue4）

#### 3.3.2 LED状态恢复
- 双闪完成后，恢复LED4到用户设置的颜色
- 恢复时考虑亮度设置（bright）

### 3.4 音效配合（可选功能）

#### 3.4.1 音效类型
- 短促的引擎启动音（约300ms）
- 或者简单的"哔"声提示

#### 3.4.2 音效时机
- 在第2次闪烁时播放
- 或者在双闪完成后播放

## 4. 技术约束

### 4.1 硬件约束
- **LED控制**: 使用WS2812B_SetAllLEDs()和WS2812B_Update()
- **尾灯编号**: LED4
- **延时函数**: HAL_Delay()（阻塞式延时）

### 4.2 性能约束
- 双闪总时长不超过1秒
- 不影响系统初始化速度
- 不占用过多Flash空间（< 100字节）

### 4.3 兼容性约束
- 不影响现有的LED控制逻辑
- 不影响用户的颜色设置
- 不影响其他LED灯带（LED1/2/3）

## 5. 非功能需求

### 5.1 用户体验
- 双闪效果清晰可见
- 节奏感强，有赛车启动的感觉
- 不会让用户感到等待时间过长

### 5.2 可维护性
- 代码结构清晰，易于理解
- 参数可配置（颜色、次数、频率）
- 易于扩展（可添加其他开机动画）

### 5.3 可靠性
- 双闪不会导致系统崩溃
- 状态恢复可靠，不会丢失用户设置
- 在任何情况下都能正常完成

## 6. 边界条件

### 6.1 首次开机
- 如果Flash中没有保存的颜色设置，使用默认颜色
- 默认颜色：红色 RGB(255, 0, 0)

### 6.2 低电量开机
- 双闪正常执行，不受电量影响
- 如果电量过低，可以考虑跳过双闪（可选）

### 6.3 快速重启
- 每次开机都执行双闪
- 不区分冷启动和热启动

## 7. 测试场景

### 7.1 基本功能测试
1. 正常开机 → 观察双闪效果
2. 双闪颜色 → 确认为纯红色
3. 双闪次数 → 确认为3次
4. 双闪时长 → 确认约1秒

### 7.2 状态恢复测试
1. 设置LED4为蓝色 → 重启 → 双闪后恢复为蓝色
2. 设置LED4为绿色 → 重启 → 双闪后恢复为绿色
3. 关闭LED4 → 重启 → 双闪后恢复为关闭

### 7.3 兼容性测试
1. 双闪时其他LED（1/2/3）状态不变
2. 双闪不影响LCD显示
3. 双闪不影响蓝牙连接

### 7.4 音效测试（可选）
1. 双闪配合音效 → 确认同步
2. 音效音量适中
3. 音效不影响双闪
