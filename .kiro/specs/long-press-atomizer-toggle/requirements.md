# 长按开启雾化器功能需求文档

## 1. 功能概述

在硬件端的speed界面（UI1）添加长按旋钮开启/关闭雾化器的功能。该功能仅在**非油门模式**下生效，不影响油门模式下的长按加速逻辑。

## 2. 用户故事

### 2.1 作为用户，我希望在speed界面长按旋钮可以快速开关雾化器
**验收标准：**
- 在UI1界面且非油门模式（wuhuaqi_state != 2）时，长按旋钮800ms可以切换雾化器状态
- 雾化器状态在开启（1）和关闭（0）之间切换
- LCD显示实时更新雾化器状态图标
- GPIO PB8引脚根据状态正确控制（开启=SET，关闭=RESET）

### 2.2 作为用户，我希望油门模式下的长按加速功能不受影响
**验收标准：**
- 在油门模式（wuhuaqi_state == 2）下，长按旋钮仍然执行加速功能
- 油门模式的所有现有逻辑保持不变
- 旋转编码器退出油门模式的功能不受影响
- 三击退出油门模式的功能不受影响

### 2.3 作为用户，我希望雾化器状态能够正确保存和恢复
**验收标准：**
- 进入油门模式前，当前雾化器状态保存到 wuhuaqi_state_saved
- 退出油门模式后，雾化器状态恢复到进入前的状态
- 长按切换的雾化器状态能够被正确保存到 wuhuaqi_state_saved

### 2.4 作为用户，我希望蓝牙控制雾化器的功能不受影响
**验收标准：**
- APP端通过WUHUA命令控制雾化器仍然正常工作
- 本地长按和远程控制的雾化器状态保持同步
- 状态变化时正确上报到APP端

## 3. 技术约束

### 3.1 现有代码结构
- 雾化器状态变量：`wuhuaqi_state`（0=关闭，1=开启，2=油门模式）
- 雾化器状态保存：`wuhuaqi_state_saved`（用于油门模式进入/退出时保存状态）
- GPIO控制：`HAL_GPIO_WritePin(GPIOB, GPIO_PIN_8, state)`
- LCD显示函数：`lcd_wuhuaqi(wuhuaqi_state, speed_value)`

### 3.2 按键检测逻辑
- 长按检测阈值：800ms（与现有代码一致）
- 按键状态变量：`key_now`, `key_old`, `key_tick`
- 当前长按检测位置：`Encoder()` 函数中，约1278行

### 3.3 不可修改的逻辑
- 油门模式（wuhuaqi_state == 2）的所有逻辑必须保持不变
- 三击进入油门模式的功能不能修改
- 双击返回菜单的功能不能修改
- 单击切换单位的功能不能修改

## 4. 边界条件

### 4.1 界面限制
- 功能仅在 `ui == 1` 且 `chu == 2` 时生效
- 在菜单界面（ui == 5）或其他界面不响应长按切换雾化器

### 4.2 模式互斥
- 油门模式下，长按=加速（现有逻辑）
- 非油门模式下，长按=切换雾化器（新增逻辑）
- 两种模式通过 `wuhuaqi_state != 2` 条件区分

### 4.3 状态同步
- 长按切换雾化器后，需要同步更新：
  - `wuhuaqi_state` 和 `wuhuaqi_state_old`
  - `wuhuaqi_state_saved`（用于油门模式恢复）
  - GPIO PB8 引脚状态
  - LCD显示

## 5. 非功能需求

### 5.1 响应时间
- 长按800ms后立即响应，延迟不超过50ms

### 5.2 用户反馈
- 状态切换后LCD图标立即更新
- GPIO控制立即生效

### 5.3 代码质量
- 代码逻辑清晰，注释完整
- 不破坏现有代码结构
- 遵循现有代码风格

## 6. 测试场景

### 6.1 基本功能测试
1. 在UI1界面，雾化器开启状态，长按800ms → 雾化器关闭
2. 在UI1界面，雾化器关闭状态，长按800ms → 雾化器开启
3. 长按过程中松开（<800ms）→ 不触发切换

### 6.2 油门模式测试
1. 三击进入油门模式 → 长按加速正常工作
2. 油门模式下长按 → 不切换雾化器状态，只加速
3. 退出油门模式 → 雾化器状态恢复到进入前的状态

### 6.3 状态保存测试
1. 雾化器开启 → 长按关闭 → 三击进入油门模式 → 退出 → 雾化器仍然关闭
2. 雾化器关闭 → 长按开启 → 三击进入油门模式 → 退出 → 雾化器仍然开启

### 6.4 蓝牙控制测试
1. APP发送WUHUA:1 → 雾化器开启 → 本地长按 → 雾化器关闭
2. APP发送WUHUA:0 → 雾化器关闭 → 本地长按 → 雾化器开启

## 7. 实现优先级

**P0（必须实现）：**
- 非油门模式下长按切换雾化器
- 油门模式下长按加速不受影响
- GPIO和LCD状态正确更新

**P1（应该实现）：**
- 状态正确保存到 wuhuaqi_state_saved
- 与蓝牙控制保持同步

**P2（可以实现）：**
- 上报状态变化到APP端（通过BLE_ReportButtonEvent或新增命令）
