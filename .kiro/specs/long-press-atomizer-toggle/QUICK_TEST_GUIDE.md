## 测试1: 基本切换功能 (2分钟)

### 步骤1: 开启→关闭
```
1. 确认雾化器当前是开启状态（LCD图标显示开启）
2. 长按旋钮800ms
3. 松开
```
**✅ 预期**: 雾化器关闭，LCD图标变为关闭状态

### 步骤2: 关闭→开启
```
1. 确认雾化器当前是关闭状态
2. 长按旋钮800ms
3. 松开
```
**✅ 预期**: 雾化器开启，LCD图标变为开启状态

### 步骤3: 短按不触发
```
1. 按下旋钮
2. 持续500ms（<800ms）
3. 松开
```
**✅ 预期**: 雾化器状态不变，可能触发单击功能（切换单位）

## 🐛 常见问题排查

### 问题1: 长按没有反应
**可能原因**:
- 不在UI1界面（检查是否在speed界面）
- 按键时间不足800ms（尝试按更久一点）
- 在油门模式下（先退出油门模式）

**解决方法**: 确认在UI1界面，长按至少800ms

### 问题2: 油门模式下切换了雾化器
**可能原因**: 代码逻辑错误

**解决方法**: 检查代码中的条件判断 `wuhuaqi_state != 2`

### 问题3: 退出油门模式后状态不对
**可能原因**: `wuhuaqi_state_saved` 未正确更新

**解决方法**: 检查切换雾化器时是否同时更新了 `wuhuaqi_state_saved`
