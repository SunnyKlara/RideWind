## 2. 模块设计

### 2.1 APP端模块

#### 2.1.1 LogoUploadScreen (UI界面)

**文件**: `lib/screens/logo_upload_screen.dart`

**职责**:
- 用户交互界面
- 图片选择与预览
- 上传进度显示
- 状态提示与错误处理

**状态管理**:
```dart
class _LogoUploadScreenState extends State<LogoUploadScreen> {
  File? _selectedImage;              // 选中的图片
  bool _isUploading = false;         // 是否正在上传
  double _uploadProgress = 0.0;      // 上传进度 0.0-1.0
  String _uploadStatus = '';         // 状态文字
  bool _hasCustomLogo = false;       // 设备是否有自定义Logo
  String _debugInfo = '';            // 调试信息
  
  StreamSubscription<String>? _rawDataSub;  // 蓝牙数据监听
}
```

**关键方法**:
```dart
// 选择图片来源
Future<void> _showImageSourceDialog();

// 从相册选择
Future<void> _pickImage(ImageSource.gallery);

// 拍照获取
Future<void> _pickImage(ImageSource.camera);

// 裁剪图片
Future<void> _cropImage(String imagePath);

// 上传Logo
Future<void> _uploadLogo();

// 恢复默认
Future<void> _restoreDefaultLogo();

// 查询Logo状态
Future<void> _checkCustomLogo();

// 测试通信
Future<void> _testCommunication();
```
