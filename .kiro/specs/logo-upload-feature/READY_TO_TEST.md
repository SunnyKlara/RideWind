## 🐛 如果遇到问题

### 问题1: 硬件无响应

**症状**: 点击"测试通信"显示超时

**解决方案**:
1. 确认固件已重新编译并烧录
2. 检查蓝牙连接状态
3. 查看硬件串口输出（printf调试信息）

### 问题2: 编译错误

**常见错误**:
- `undefined reference to 'LCD_Writ_Bus'` → 检查logo.c中的extern声明
- `undefined reference to 'Logo_ShowBoot'` → 检查lcd.c中的#include "logo.h"
- `gImage_tou_xiang_154_154 undeclared` → 检查pic.h中的数组定义

### 问题3: 上传失败

**可能原因**:
- CRC校验失败 → 重新上传
- 蓝牙信号弱 → 靠近设备
- Flash写入失败 → 检查硬件
