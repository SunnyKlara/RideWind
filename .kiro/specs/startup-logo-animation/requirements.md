# 开机Logo动画功能需求

## 概述
将当前静态的开机Logo图片替换为动态几何动画，展示正方形、对角线和内切圆的优雅几何关系。

## 用户故事

### US-1: 动态开机动画
作为用户，我希望设备开机时显示一个动态的几何动画，而不是静态图片，让开机体验更加生动有趣。

#### 验收标准
- AC-1.1: 动画首先绘制正方形轮廓
- AC-1.2: 对角线从正方形中心点向两端延伸
- AC-1.3: 圆形与对角线、下边框、右边框同时相切
- AC-1.4: 动画流畅，无明显卡顿
- AC-1.5: 动画总时长控制在1-2秒内

### US-2: 几何关系正确性
作为用户，我希望动画中的几何元素关系正确，体现数学美感。

#### 验收标准
- AC-2.1: 正方形居中显示在240x240屏幕上
- AC-2.2: 对角线精确连接正方形的两个对角
- AC-2.3: 内切圆与对角线外切，与下边框和右边框内切
- AC-2.4: 所有几何元素颜色协调（建议白色线条，黑色背景）

### US-3: 动画可配置性
作为开发者，我希望动画参数可以方便调整。

#### 验收标准
- AC-3.1: 正方形大小可配置
- AC-3.2: 动画速度可配置
- AC-3.3: 线条颜色可配置

## 技术约束
- 硬件平台：STM32F4微控制器
- 显示屏：240x240 LCD圆形屏
- 现有绘图API：LCD_DrawLine, Draw_Circle, LCD_DrawRectangle, LCD_Fill
- 代码位置：logo.c 中的 Logo_ShowBoot() 函数

## 几何数学关系

设正方形边长为 `a`，左上角坐标为 `(x0, y0)`：

1. **正方形**：四个顶点为 (x0, y0), (x0+a, y0), (x0, y0+a), (x0+a, y0+a)

2. **对角线**：从 (x0, y0) 到 (x0+a, y0+a)

3. **内切圆**：
   - 圆心在对角线的垂直方向偏移
   - 设圆心为 (cx, cy)，半径为 r
   - 与下边框相切：cy + r = y0 + a
   - 与右边框相切：cx + r = x0 + a
   - 与对角线相切（外切）：点到直线距离 = r
   
   对角线方程：y - y0 = x - x0，即 x - y + (y0 - x0) = 0
   
   圆心到对角线距离：|cx - cy + (y0 - x0)| / √2 = r
   
   由于圆在对角线右下方：cx > cy（相对于对角线）
   
   解方程组可得：
   - r = a * (√2 - 1) / (√2 + 1) ≈ 0.172 * a
   - cx = x0 + a - r
   - cy = y0 + a - r
