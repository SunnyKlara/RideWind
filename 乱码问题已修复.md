## 技术细节

### RGB565格式
每个像素2字节:
```
位15-11: 红色 (5位, 0-31)
位10-05: 绿色 (6位, 0-63)
位04-00: 蓝色 (5位, 0-31)
```

### 转换过程
```dart
// 1. 调整图片到240x240
final resized = img.copyResize(image, width: 240, height: 240);

// 2. 转换每个像素
for (int y = 0; y < 240; y++) {
  for (int x = 0; x < 240; x++) {
    final pixel = resized.getPixel(x, y);
    
    // 提取RGB分量并转换为5/6/5位
    final r5 = (pixel.r >> 3) & 0x1F;  // 8位→5位
    final g6 = (pixel.g >> 2) & 0x3F;  // 8位→6位
    final b5 = (pixel.b >> 3) & 0x1F;  // 8位→5位
    
    // 组合成RGB565
    final rgb565 = (r5 << 11) | (g6 << 5) | b5;
    
    // 大端序存储
    result[index++] = (rgb565 >> 8) & 0xFF;
    result[index++] = rgb565 & 0xFF;
  }
}
```
