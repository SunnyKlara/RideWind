# 🔴 Logo上传失败 - 诊断方案

## 当前状况

- ❌ 主界面显示默认Logo（头像）
- ⚠️ LCD显示 "Logo Debug Mode"
- ✅ 发送了7200包数据（115200字节）
- ✅ LCD显示了 "END CMD RCV"

**结论**：数据传输完成，但验证失败或Header未写入。

## 🔍 可能的原因

### 原因1：CRC校验失败（最可能）
- 数据已写入Flash
- 但CRC32不匹配
- Header未写入
- 显示默认Logo

### 原因2：大小校验失败
- 某些包丢失
- 接收大小不等于115200
- 验证失败

### 原因3：Flash写入失败
- 数据未写入Flash
- Flash全是0xFF
- CRC校验失败

## 📋 诊断步骤

### 步骤1：查看LCD调试信息

**当前LCD显示 "Logo Debug Mode"，说明：**
- 硬件还在UI6（Logo调试界面）
- 没有自动切换回主界面
- 可能验证失败后停在了调试界面

**操作**：
1. 在APP中发送 `UI:1` 命令
2. 切换回主界面
3. 看是否有变化

### 步骤2：发送GET:LOGO命令

这个命令比LOGO_TEST简单，更容易成功。

**在APP中发送**：
```
GET:LOGO
```

**预期响应**：
- `LOGO:1` - 有自定义Logo
- `LOGO:0` - 无自定义Logo

**如果响应是 `LOGO:0`**：
- 说明Header无效
- Logo_IsValid()返回false
- 验证失败

### 方案A：添加详细的串口日志

修改 `logo.c` 的 LOGO_END处理器，添加更多日志：

```c
else if (strcmp(cmd, "LOGO_END") == 0)
{
    Logo_AddDebugLog("END CMD RCV");
    
    if (logo_state != LOGO_STATE_RECEIVING) {
        printf("[LOGO] ERROR: Not in receiving state for END\r\n");
        BLE_SendString("LOGO_ERROR:NOT_RECEIVING\n");

    // 🔥 添加详细日志

    printf("[LOGO] ═══════════════════════════════════\r\n");

    if (logo_received_size != logo_total_size) {

                (unsigned long)logo_received_size, 
                (unsigned long)logo_total_size);
        BLE_SendString(response);
        logo_state = LOGO_STATE_ERROR;
        return;
    }
    
    printf("[LOGO] ✓ Size check passed\r\n");
    Logo_AddDebugLog("SIZE OK");
    
    // 计算CRC32
    printf("[LOGO] Calculating CRC32...\r\n");
    Logo_AddDebugLog("CALC CRC...");
    
    uint32_t calculatedCRC = CRC32_CalculateFlash(
        LOGO_FLASH_ADDR + LOGO_HEADER_SIZE, 
        logo_total_size
    );
    
    printf("[LOGO] CRC32 calculated: 0x%08lX\r\n", (unsigned long)calculatedCRC);
    printf("[LOGO] CRC32 expected:   0x%08lX\r\n", (unsigned long)logo_expected_crc);
    
    if (calculatedCRC != logo_expected_crc) {
        printf("[LOGO] ❌ CRC MISMATCH!\r\n");
        Logo_AddDebugLog("CRC FAIL!");
        
        // 🔥 读取Flash前16字节用于调试
        uint8_t debug_data[16];
        W25Q128_BufferRead(debug_data, LOGO_FLASH_ADDR + LOGO_HEADER_SIZE, 16);
        
        printf("[LOGO] First 16 bytes from Flash:\r\n");
        printf("[LOGO]   ");
        for (int i = 0; i < 16; i++) {
            printf("%02X ", debug_data[i]);
        }
        printf("\r\n");
        
        // LCD显示CRC错误
        LCD_Fill(0, 0, 240, 240, BLACK);
        LCD_ShowString(40, 60, (u8*)"CRC ERROR", RED, BLACK, 24, 0);
        char msg[40];
        snprintf(msg, 40, "EXP:0x%08lX", (unsigned long)logo_expected_crc);
        LCD_ShowString(20, 100, (u8*)msg, WHITE, BLACK, 16, 0);
        snprintf(msg, 40, "GOT:0x%08lX", (unsigned long)calculatedCRC);
        LCD_ShowString(20, 130, (u8*)msg, WHITE, BLACK, 16, 0);
        
        // 显示Flash前16字节
        LCD_ShowString(20, 170, (u8*)"FIRST 16 BYTES:", CYAN, BLACK, 12, 0);
        for (int i = 0; i < 16; i += 4) {
            snprintf(msg, 40, "%02X %02X %02X %02X", 
                    debug_data[i], debug_data[i+1], debug_data[i+2], debug_data[i+3]);
            LCD_ShowString(20, 185 + i*4, (u8*)msg, GREEN, BLACK, 12, 0);
        }
        
        sprintf(response, "LOGO_FAIL:CRC:0x%08lX\n", (unsigned long)calculatedCRC);
        BLE_SendString(response);
        logo_state = LOGO_STATE_ERROR;
        return;
    }
    
    printf("[LOGO] ✓ CRC check passed\r\n");
    Logo_AddDebugLog("CRC OK");
    
    // 写入Header
    printf("[LOGO] Writing header...\r\n");
    Logo_AddDebugLog("WRITE HDR...");
    
    LogoHeader_t header = {
        .magic = LOGO_MAGIC,
        .width = LOGO_WIDTH,
        .height = LOGO_HEIGHT,
        .reserved1 = 0,
        .dataSize = logo_total_size,
        .checksum = calculatedCRC
    };
    W25Q128_BufferWrite((uint8_t*)&header, LOGO_FLASH_ADDR, sizeof(header));
    
    printf("[LOGO] ✓ Header written\r\n");
    Logo_AddDebugLog("SUCCESS!");
    
    // LCD显示成功
    LCD_Fill(0, 0, 240, 240, BLACK);
    LCD_ShowString(60, 80, (u8*)"SUCCESS!", GREEN, BLACK, 32, 0);
    LCD_ShowString(40, 140, (u8*)"LOGO UPLOADED", WHITE, BLACK, 20, 0);
    char msg[40];
    snprintf(msg, 40, "%lu bytes", (unsigned long)logo_total_size);
    LCD_ShowString(70, 180, (u8*)msg, CYAN, BLACK, 16, 0);
    
    logo_state = LOGO_STATE_COMPLETE;
    BLE_SendString("LOGO_OK\n");
    
    printf("[LOGO] ═══════════════════════════════════\r\n");
    printf("[LOGO] ✅ Upload complete!\r\n");
    printf("[LOGO] ═══════════════════════════════════\r\n");
```
