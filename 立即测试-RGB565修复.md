## 技术细节

### RGB565格式
每个像素2字节:
- R: 5位 (红色)
- G: 6位 (绿色)
- B: 5位 (蓝色)
- 大端序存储

### 转换过程
```dart
// 1. 调整图片大小到240x240
final resized = img.copyResize(image, width: 240, height: 240);

// 2. 转换每个像素到RGB565
for (int y = 0; y < 240; y++) {
  for (int x = 0; x < 240; x++) {
    final pixel = resized.getPixel(x, y);
    final r5 = (pixel.r >> 3) & 0x1F;
    final g6 = (pixel.g >> 2) & 0x3F;
    final b5 = (pixel.b >> 3) & 0x1F;
    final rgb565 = (r5 << 11) | (g6 << 5) | b5;
    // 大端序存储
    result[index++] = (rgb565 >> 8) & 0xFF;
    result[index++] = rgb565 & 0xFF;
  }
}
```
